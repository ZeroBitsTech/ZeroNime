## [0.1.0] - 2026-03-17

### Added

- Initial public release of the ZeroNime backend.
- `api/v2` backend for anime catalog browsing, stream resolution, and playback-oriented media proxying.
- Startup media cache with local-first storage and optional DObject backing store.
- Predictive next-episode startup cache for sequential playback.
